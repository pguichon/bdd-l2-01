La note de QCM : je peux retirer la moins bonne et la meilleur (sur demande de l'étudiant)

  • Si négative : la différence avec 10 est retirer de la note du projet
  • Si positive : la différence avec 10 diviser par deux est ajouter à la note du TP
Exemple :
  • J'ai 8 de moyenne pour le QCM : 10 - 8 = 2 : -2 points
  • J'ai 6 à mon TP : la note totale 6 - 2 = 4
  • J'ai 14 à mon TP : la note totale 14 - 2 = 12
  • J'ai 14 de moyenne : (14 - 10) / 2 = 2
  • J'ai 8 à mon TP : 8 + 2 = 10
  • J'ai 14 à mon TP : 14 + 2 = 16

Le SQL

SQL : Structured Query Langage

C'est un langage de la forme Verbe (action) sur (élément) d'ou (table) condition

Table

Les données sont stockées dans des tables

Ligne de la table correspond à un enregistrement

Une colonne correspond à une propriété de l'enregistrement

Exemple

id code label quantity
1 0001 Marteau 10
2 0002 Tournevis 3
3 0003 Clous 1000

Primary Key

  • Il s'agit d'une colonne qui contiendra une valeur unique
  • Généralement on utilise un nom et on nomme cette colonne id
  • Lorsque c'est possible on utilisera un champ auto-incrémenté

Création d'une table

						
							CREATE TABLE articles (
							id INTEGER PRIMARY KEY,
							code TEXT,
							label TEXT,
							quantity INTEGER);

Création d'une table

						
							CREATE TABLE article s(
							id INTEGER,
							code TEXT,
							label TEXT,
							quantity INTEGER,
							PRIMARY KEY(id));

Suppression d'une table


						DROP TABLE articles;

Selection


						SELECT id, code, clabel, quantity FROM articles;
Avec le * 

						SELECT * FROM articles;
Attention si * l'ordre des colonnes est aléatoire

Selection avec condition 


						SELECT id, code, clabel, quantity FROM articles WHERE id=1;
